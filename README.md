## Development

Install any missing packages from your machine:
```npm install -g gatsby-cli```
```npm install```

Run gatsby in develop mode (hot-preview changes):
```gatsby develop```

Deploy to surge.sh:
```npm install -g surge```
```gatsby build```
```surge public```
